Tratamento de Dados

Projeto feito em JavaScript, seu objetivo foi manejar banco de dados e graficos. Sua primeira etapa se trata de uma pesquisa ficticia realizada com IA sobre as Maquiagens mais consumidas. A segunda etapa se trata de uma pesquisa de campo feita na escolei em que estudei CCM-Tancredo de Almeida Neves tendo semelhança com a pesquisa ficticia.

Acerte o numero

Projeto desenvolvido em JavaScript utilizando uma API para reconhecimento de voz. Seu objetivo é permitir a captura de voz. Desta forma, ao falar um número o site ira dizer se o valor dito é menor ou maior do que o némero secreto. Se a API identificar uma fala diferente de um valor numeral, entregara uma mensagem dizendo que o que foi dito é inválido.
